# MiLight (easybulb) app for Homey  
Connect your MiLight (EasyBulb) light bulbs with Homey.

Homey connects with your WiFi bridge to control the lights.

Also compatible with:
- Rocket LED
- Limitless LED Applamp
- s`luce
- iLight
- iBulb
- Kreuzer
